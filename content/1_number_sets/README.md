# Number Sets

- Natural: $\mathbb{N}$
- Whole: $\mathbb{W}$
- Real: $\mathbb{R}$
- Rational: $\mathbb{Q}$
- Irrational: $\mathbb{I}$
- Imaginary: NA (not available)
- Complex: $\mathbb{C}$
